# Volatility Spillovers Across Global Asset Classes
Master's Thesis — Tom Jonkers | HSLU MSc Banking & Finance | 2026
Supervisor: Denis Bieri

## Overview
This repository contains the Python notebooks used for the empirical 
analysis. Three model specifications are included:
- Model 1: MSCI World, Gold, Brent, Bitcoin, US10Y, EU10Y
- Model 2: Model 1 + Ethereum
- Model 3: Bitcoin and Ethereum only

All models are estimated over a full sample (2015–2025) and three 
subperiods: 2015–2018, 2019–2021, and 2022–2025.
